# InAmigos Foundation NGO Awareness Webpage

A responsive NGO awareness webpage created for the InAmigos Foundation awareness task.

## 📌 Project Objective

This webpage was developed to spread awareness about the initiatives, projects, and social impact of InAmigos Foundation through a clean and responsive design.

## ✨ Features

- Responsive Navigation Bar
- Mobile Hamburger Menu
- Hero Image Slider
- NGO Program Highlights
- About Section
- Ongoing Projects Section
- Social Impact Section
- Campaign Gallery
- Volunteer Call-To-Action
- Responsive Design for Mobile, Tablet & Desktop
- Smooth CSS Animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
